- **Repository name**: `malharia-queiroz`
- **Description**: `Aplicativo web para design de camisas e uniformes`
- **Visibility**: Public ✅

### Passo 2️⃣: Executar no Terminal

```bash

```

### Passo 3️⃣: Fazer Push
