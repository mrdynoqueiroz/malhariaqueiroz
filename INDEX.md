## 🚀 malharia-queiroz.html

**Tipo**: Aplicativo web (HTML + CSS + JS)
**Tamanho**: 58KB
**Status**: ✅ Pronto para produção

### O que é?
Seu aplicativo web completo para design de camisas. Um arquivo único contendo:
- Designer interativo com canvas
- 8 cores de camisa
- Upload de imagem
- Posicionamento dinâmico
- Controle de texto
- Exportação PNG
- Compartilhamento WhatsApp

### Como usar?
```bash
python3 -m http.server 8000
# Abrir: http://localhost:8000/malharia-queiroz.html
```

### Necessário?
- **Sim!** Este é o aplicativo principal
